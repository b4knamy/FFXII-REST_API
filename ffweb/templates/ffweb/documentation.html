FFXII API Documentation

This API is free without request limit, you can take how many times you want but it just support until 2 parameters, we'll see more about later.

Data options

First of all, you put that URL "domain/api/", after that, you have two choices: getting "Espers" or "Characters" data. 

                    "http://127.0.0.1:8000/api/characters"

or

                    "http://127.0.0.1:8000/api/espers"

Getting All Data

To get all data, you put the keyword "/all" for characters or espers, this is totally up to you, it will look like this: 

                    "http://127.0.0.1:8000/api/characters/all"

Getting especific data

Now, to get an especific data, you can use "/filter" instead, but you have to set a parameter, which depends of what you want.
For example, lets filter using a name: 

                    "http://127.0.0.1:8000/api/characters/filter?name=vaan"

Parameter "limit"

As this is a small project, FFXII API just allow GET requests with only two parameters. The "limit" parameter will always be the last parameter, if you are using two parameters, it will look like this: 

                    "http://127.0.0.1:8000/api/characters/filter?hometown=archades&limit=3"

otherwise, if you want all the datas for example, you can just use "limit".

                    "http://127.0.0.1:8000/api/espers/all?limit=3"

That's all for now, I hope you liked!
